# Changelog

All notable changes to this project will be documented in this file.

## 0.0.1 - 2026-04-24

- Initial release.
- Added `Go to Definition` support for PascalCase component tags in `.mdx` files.
- Supports default import resolution for relative paths and `@/` aliases.
- Supports target files: `.astro`, `.vue`, `.tsx`, `.ts`, `.jsx`, `.js`.
- Added import CodeLens: `Open <ComponentName>`.
